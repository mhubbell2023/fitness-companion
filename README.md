# Fitness Companion App

## Overview
The Fitness Companion App is a web-based application that allows users to track their workouts, view their workout history, and manage their fitness progress. The app provides a simple and user-friendly interface for logging exercises and monitoring performance over time.

## Features
- User registration and login
- Secure authentication
- Add and track workouts
- View workout history
- Clean and responsive UI

## Technologies Used
- Python (Flask)
- SQLite
- HTML/CSS
- GitHub Actions (CI/CD)
- Microsoft Azure App Service (Deployment)

## How It Works
1. Users create an account and log in
2. Users can add workouts with details like exercise, sets, reps, and weight
3. The app stores data in a SQLite database
4. Users can view and edit their workout history on the dashboard

## CI/CD Pipeline
This project uses GitHub Actions for continuous integration and deployment.

- Code is pushed to GitHub
- GitHub Actions runs the workflow
- The application is automatically built and deployed to Azure
